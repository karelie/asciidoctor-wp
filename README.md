# asciidoctor-wp
Asciidoctor for Wordpress

## Description

This Plugin is Asciidoc markups plugin for wordpress.

## Installation

1. Upload `asciidoctor-wp` to the `/wp-content/plugins/` directory
2. Activate the plugin through the ‘Plugins’ menu in WordPress
